# 🚀 OfficeDesk: Dealership Utility Suite  

A modern, responsive web-based toolkit designed for dealerships to streamline warranty PPTs, service reports, and more. Built with **HTML/CSS/JS**, **Firebase**, and **service workers** for offline support.  

---

## 🌟 Key Features  

### 🔧 Tools & Functionality  
- **Multiple Forms**:  
  - Alcazar Form (PPT)  
  - Key Code Form (XLS)  
  - Letter Heads (DOCX)  
  - PWA Regular (XLSX)  
  - ETM Form (PPT)  
  - AGM Form (XLSX)  
- **Profile Management**:  
  - Customizable avatar (upload or choose from templates)  
  - Nickname settings  
- **Backup & Restore**:  
  - **Local Backup**: Save data to a `.json` file  
  - **Cloud Backup**: Securely sync to Firebase  
- **Theme Switching**:  
  - Light/Dark mode toggle  
- **Offline Support**:  
  - Service worker for offline functionality  
- **Responsive Design**:  
  - Works on all screen sizes (mobile, tablet, desktop)  

---

## 🛠️ Installation  

1. **Clone or Download**:  
   - Clone the repository or download the `index.html` file.  
2. **Set Up Firebase**:  
   - Use the provided Firebase config (`firebaseConfig.js`) to initialize Firebase.  
3. **Open in Browser**:  
   - Open `index.html` in any modern browser.  

---

## 🧪 Usage  

1. **Dashboard**:  
   - Navigate to the dashboard to access tools.  
2. **Tools**:  
   - Click on any tool (e.g., Alcazar Form) to open the corresponding iframe.  
3. **Profile**:  
   - Customize your avatar and nickname via the profile menu.  
4. **Backup/Restore**:  
   - Use local or cloud backup options.  
5. **Theme**:  
   - Toggle between light and dark modes.
